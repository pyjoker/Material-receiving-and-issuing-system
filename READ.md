# Material Receiving and Issuing System - Excel Reader

物料收發系統的 Excel 檔案讀取模組

## 功能特色

- ✅ 讀取多工作表 Excel 檔案（.xlsx, .xls）
- ✅ 支援工作表名稱和索引兩種方式
- ✅ 返回 pandas DataFrame 格式
- ✅ 資料預處理功能（刪除空列/空欄、填充空值）
- ✅ 完整的錯誤處理
- ✅ 支援 with 語句自動管理資源

## 安裝

1. 確保已安裝 Python 3.8 或更高版本

2. 安裝依賴套件：

```bash
pip install -r requirements.txt
